Hi there! We’re thrilled that you’d like to contribute to the Government Open Source Software Policies Dataset. Be sure to read our README - it describes the project and its purpose.

We welcome two types of contributions: - Pull Requests: If you’d like to add a new policy or amend and existing policy within the dataset, please open a PR. - Issues: If you’d like to add a new criterion or category for all policies, please open an issue to facilitate discussion as the first step.

Contributions will then be periodically reviewed for updates to the dataset.

Contributions to this project are subject to the CC BY-4.0 license.

Please note that this project is released with a Contributor Code of Conduct. By participating in this project you agree to abide by its terms.
